### マルチログインの応用例

| サイトの種類 |        提供側(販売側)        |  利用側(購入側)  |
| :----------: | :--------------------------: | :--------------: |
|     物販     |          商品の登録          | 商品を探す・買う |
|    不動産    |          物件の登録          |    物件を探す    |
|     求人     |        求人情報の登録        |  求人情報を探す  |
|     副業     |         スキルの登録         |     依頼する     |
|   家電修理   | エアコンなどの修理内容を登録 |  探す・依頼する  |
